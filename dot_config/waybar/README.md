Custom cyberpunk-like style waybar

> Preview :
> ![preview](./../../assets/waybar_preview.png)


Includes:
- 🕔 Time/Date
- 🌡️CPU, GPU and RAM usage (with [gpu-usage-waybar](https://github.com/PolpOnline/gpu-usage-waybar))
- 🖥️ Primary and Secondary monitor hyprland workspace id display
- 🔊 Music control (audio output, volume, media pause/play/prev/next)
- 🎶 Song title
- 🔋Power menu
- ⛅ Weather (with [waybar-weather](https://github.com/wneessen/waybar-weather/))
- 📱Kdeconnect Status with (with [kdeconnect_waybar](https://github.com/Adrien5902/kdeconnect_waybar))
